# react-native-skia-loading

High-performance loading animations for React Native using Skia

## Installation


```sh
npm install react-native-skia-loading
```


## Usage


```js
import { SkiaLoadingView } from "react-native-skia-loading";

// ...

<SkiaLoadingView color="tomato" />
```
